Git Hub Instructions

Go to Git Hub and signup for an account

Go through all the information and create a new repository. It will have a clone address that you will want to copy.

Download Git by going to : https://git-scm.com/ and downloading the version for windows.

Install that and you should be ready to connect your github to VSCode.

Restart VS Code to make sure that Git is recognized in the terminal.

Make sure your folder is opened in VS Code. Then Click Terminal at the top and select New Terminal

type in git clone and then right click after you have the address copied from your repository.

then type in git config --global user.name "YourUserName"

then type in git config --global user.email "YourUserEmail"

This should link your account to the your repository.

On the left side of VSCode you should see the source control icon. Looks like 3 dots that are conencted.

This will track all of your changes and this is where you can push or pull files to github so that they can be accessed anywhere.
